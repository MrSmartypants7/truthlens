Pipeline

Four steps, zero hallucinations.

Every claim in an LLM response is individually extracted, embedded, retrieved against ground-truth knowledge, and scored with confidence.

📝
Claim Extraction

LLM breaks the response into individual, verifiable factual claims

🔎
Evidence Retrieval

Each claim is embedded and searched against the FAISS vector index

⚖️
Cross-Verification

Secondary LLM prompt evaluates each claim against retrieved evidence

📊
Reliability Score

Claims are classified with confidence scores and an overall score is computed

Features

Production-ready, out of the box.

Every layer — from ingestion to scoring — is designed for reliability and extensibility.

⚡
Batched FAISS Search

All claims are embedded and searched in a single batched call — not N sequential queries. Sub-300ms for cached queries.

🧠
Secondary Verification

Claims are verified against retrieved external evidence — not self-verified by the same model that generated them.

🔒
Fully Local

Runs on Ollama with llama3.2 and nomic-embed-text. No API keys, no cloud calls, no cost per request.

📦
Embedding TTL Cache

Repeated queries hit cache, saving ~200ms per call. Cache size and TTL are configurable via environment variables.

📈
Experiment Tracking

Built-in benchmark logging with accuracy, precision, and recall across models. Queryable via /experiments.

🪵
Structured JSON Logging

Production-grade observability with structlog. CloudWatch and Datadog compatible out of the box.

API

Simple to integrate.

REST endpoints for verification, ingestion, benchmarking, and health checks.

POST /verify CURL 
# Verify an LLM response
curl -X POST localhost:8000/verify \
  -H "Content-Type: application/json" \
  -d '{
    "llm_response": "The Eiffel Tower
      is 324 meters tall, built in
      1889 for the World'\''s Fair."
  }'

  "overall_reliability_score": 0.82,

    "status": "supported",

    "reasoning": "Height confirmed..."

POST /ingest CURL 
# Add documents to knowledge base
curl -X POST localhost:8000/ingest \
  -H "Content-Type: application/json" \
  -d '{
    "documents": [{
      "text": "The Eiffel Tower
        is 330m, built 1887-1889.",
      "source": "wikipedia"
    }]
  }'
Quick Start BASH 
# 1. Pull models
ollama pull llama3.2
ollama pull nomic-embed-text

# 2. Install & ingest demo data
pip install -r requirements.txt
python scripts/demo.py

# 3. Start the server
uvicorn app.main:app --reload
